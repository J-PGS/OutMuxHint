硬链接(Hard link):
https://ss64.com/nt/mklink.html


Link Shell Extention(Windows平台GUI创建链接工具)：
https://schinagl.priv.at/nt/hardlinkshellext/linkshellextension.html



如果是刚开始使用外挂结构/对硬链接不熟悉，推荐 OutMuxSub_hardlink_Win10.bat 和 Link Shell Extention 手动。
OutMuxSub_hardlink_WinALL.bat 相对更为便捷，但没有提示。
分发外挂结构愿意写链接脚本可参考 hardlink_WinALL.bat 和 ln_macOS.sh 的写法。

※WinALL 已不再考虑于 WinXP 上的使用情况。如有需要请手动将第五行的 chcp 65001 去除后保存可用。
